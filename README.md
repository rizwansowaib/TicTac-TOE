# Tic-Tac-Toe Game

A lightweight, responsive, and interactive **Tic-Tac-Toe** web game built using clean vanilla web technologies. This project demonstrates basic game state management, turn-based logic handling, grid layouts, and dynamic DOM manipulation.

---

## 🚀 Features
* **Responsive Design:** Works flawlessly on mobile, tablet, and desktop viewports.
* **Smart Turn Indicator:** Real-time updates showcasing whose turn it is (Player X or Player O).
* **Automated Win/Draw Detection:** Instant verification of the board state against all 8 winning combinations.
* **Interactive Colors:** Distinct custom color coding for X and O selections to maximize UI clarity.
* **One-Click Reset:** A dedicated button to completely clear the game board and state instantly.

---

## 🛠️ Tech Stack
* **HTML5:** For the structural semantic markup of the game elements.
* **CSS3:** Styled utilizing **CSS Grid** layout to center and align the 3x3 game board.
* **JavaScript (ES6):** Handles core game arrays, win/loss algorithms, and event handling.

---

## 🎮 How to Play
1. The game starts with **Player X**.
2. Click on any empty square on the grid to place your marker.
3. The game will automatically switch turns to **Player O**.
4. The first player to align 3 of their markers horizontally, vertically, or diagonally wins the match!
5. If all 9 squares are filled without a winner, the match ends in a draw.
6. Click the **Restart Game** button to reset the grid and start a fresh match.
